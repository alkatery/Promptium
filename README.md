# 🧠 Promptium — خبير هندسة الأوامر العالمي

**Promptium** هو برومبت نظام (System Prompt) احترافي يحوّل أي نموذج ذكاء اصطناعي إلى **خبير عالمي في هندسة الأوامر (Prompt Engineering)**، قادر على تحويل أي فكرة أو طلب — مهما كان بسيطاً — إلى برومبت نصي عالي الجودة، دقيق، مفصّل، ومنظّم وجاهز للاستخدام مباشرة.

يعمل مع جميع النماذج الحالية والمستقبلية:

| نماذج المحادثة | نماذج الصور | نماذج الفيديو |
|:---|:---|:---|
| ChatGPT · Claude · Gemini · Grok · Copilot | Midjourney · Stable Diffusion · Flux · DALL·E · Ideogram · Leonardo AI | Veo · Sora · Runway · Pika |

---

## 🖥️ الواجهة التفاعلية — اكتب فكرتك واحصل على البرومبت

المستودع يتضمن **واجهة ويب كاملة** ([`index.html`](index.html)) تكتب فيها فكرتك فيحوّلها البرنامج إلى برومبت احترافي بالتنسيق الكامل، بوضعين:

- **⚡ الوضع الفوري** — محرك تحويل مدمج يعمل فوراً بدون إنترنت وبدون أي مفتاح: يكشف نوع المهمة تلقائياً (نص / صورة / فيديو / كود) ويولّد البرومبت مع Negative Prompt والإعدادات والمقاسات وأزرار نسخ لكل مخرج.
- **🤖 وضع الذكاء الاصطناعي** — للنتائج المخصصة بالكامل، بمفتاح API خاص بك (Claude أو Gemini أو ChatGPT). المفتاح يُحفظ في متصفحك فقط.

**طرق التشغيل:**

1. **مباشرة:** حمّل [`index.html`](index.html) وافتحه في أي متصفح — يعمل فوراً.
2. **عبر GitHub Pages (رابط دائم):** من **Settings ← Pages** اختر **Deploy from a branch** ثم الفرع `main` والمجلد `/ (root)` واحفظ — ستصبح الواجهة متاحة على `https://alkatery.github.io/Promptium/`.

---

## ⚡ البدء السريع (3 خطوات)

1. افتح ملف **[PROMPT.md](PROMPT.md)** وانسخ البرومبت الكامل.
2. الصقه كـ **System Prompt** أو كأول رسالة في النموذج الذي تستخدمه.
3. اكتب فكرتك أو هدفك فقط — وسيولّد النموذج برومبتاً احترافياً كاملاً وجاهزاً للنسخ.

> **مثال:** اكتب «أريد صورة لمقهى عربي تراثي» وستحصل على برومبت صور كامل مع Negative Prompt وإعدادات التصوير والإضاءة ونسخ جاهزة لجميع المقاسات ومنصات التواصل.

## 🎯 ماذا يفعل Promptium؟

عند تفعيله، يتبع النموذج منهجية صارمة من ثلاث مراحل:

1. **تحليل الطلب** — استخراج الهدف، نوع المهمة، الجمهور، المجال، واستنتاج المعلومات الناقصة.
2. **تحسين الطلب** — إزالة الغموض، إضافة السياق والقيود، وتحديد معايير الجودة والنجاح.
3. **إنشاء البرومبت النهائي** — بصيغة كاملة تشمل الدور، الهدف، التعليمات التفصيلية، القيود، تنسيق الإخراج، وآلية التحقق من النتائج.

ويكيّف مخرجاته تلقائياً حسب نوع الطلب:

- **النصوص** ← ثلاث نسخ: مختصرة، احترافية، ومتقدمة جداً.
- **الصور** ← برومبت احترافي + Negative Prompt + إعدادات التصوير والعدسة والإضاءة + نسخ جاهزة لجميع المقاسات (مربع، طولي، عرضي) ومنصات التواصل الاجتماعي.
- **الفيديو** ← وصف المشاهد، حركة الكاميرا، الإضاءة، المؤثرات، الانتقالات، الصوت، وتعليمات خاصة بـ Veo وSora وRunway وPika.
- **الكود** ← اللغة، البيئة، المتطلبات، القيود، الأداء، الأمان، الاختبارات، وشكل الإخراج.

---

## 🛡️ القواعد الصارمة المدمجة

- لا يختلق معلومات غير مؤكدة أبداً.
- يستخدم أفضل الممارسات الحديثة في هندسة الأوامر.
- يجعل البرومبتات قابلة للعمل على مختلف النماذج.
- مخرجات جاهزة للاستخدام دون تعديل كلما أمكن.
- متغيرات قابلة للتخصيص بين أقواس مثل: `[المجال]` `[الجمهور]` `[اللغة]` `[المنصة]` `[الهدف]` `[المدة]` `[نسبة الأبعاد]`.
